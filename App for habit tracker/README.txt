1-Year Daily Checker App Package

Files included:
- index.html
- manifest.json
- sw.js
- icons/

How to use:
1. Upload all files to Netlify, GitHub Pages, or any static hosting.
2. Open the hosted URL in Chrome/Edge on Android or desktop.
3. Use the browser menu and choose Install app / Add to Home Screen.

Note:
- This is a web app package (PWA), not a native Android APK.
- Local data stays in the browser storage of the device/browser where you use it.
